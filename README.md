## I'm a Consultant, a Software Engineer, a Teacher and a Speaker!

I have been working with .NET since its first introduction. Currently I work as freelance consultant and I'm involved in designing and developing applications for the Microsoft ecosystem. I organize and holds training courses on these topics. I'm co-founder and Community Leader of [DotNetToscana](https://dotnettoscana.org) and member of [DotNetLiguria](https://dotnetliguria.net), two Italian User Groups. I'm speaker at technical events and writes articles for magazines. I'm a Microsoft MVP since 2013.
